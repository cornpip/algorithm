## Python
전역 변수 함수에서 그냥 쓰는 건 가능  
전역 변수에 재할당은 런타임 에러  _( 재할당 할거면 global 쓰기 )_  
ex)
```
a = 1
def test():
    a + 1
    return a
# 1
```
재할당은 안하고 이렇게 연산만 취하면 에러는 없으나 생각대로 동작 안함 _( 주의 )_
