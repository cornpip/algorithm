### 점화식(재귀식)
수학의 점화식은 특정한 함수를 자신보다 더 작은 변수에 대한 함수와의 관계로 표현한 것을 의미  
`ex) factorial(n) = n * factorial(n-1)`  

---
그래프 표현 방식 2가지
+ 인접 행렬 : 특정 노드와 연결을 확인해야할 경우 유리
+ 인접 리스트 : 모든 인접 노드를 순회해야 할 경우 유리  

---
일반적인 경우 수행시간은 DFS보다 BFS이 좋은 편이다.

||DFS|BFS|
|--|---|---|
|동작원리| 스택| 큐
|구현방법| 재귀함수 이용| 큐 자료구조 이용(deque)
