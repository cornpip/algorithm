## 메모리 사용량

int 자료형일 때 개수에 따른 사용량  
|데이터 개수 _( 리스트 길이 )_| 메모리 사용량 |
|------|----|
|1,000| 4KB |
|1,000,000| 4MB |
| 10,000,000 | 40MB |

보통 128 ~ 512MB 제한  
크기가 1,000만 이상인 리스트가 있다면 주의 하자.  

---
python은 자료형 지정할 필요가 없어 자료형 범위 제한에 딱히 생각안해도된다.  

일반적으로 `1초에 2,000만 번의 연산을 수행한다고 가정`하면 시간 제한에 안정적이라 한다.
